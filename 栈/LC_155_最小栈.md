# LeetCode 155. 最小栈

### 1. 题目链接
[https://leetcode.cn/problems/min-stack/description/?envType=study-plan-v2&envId=top-100-liked]

### 2. 我的初次思路与遇到的问题 (The Error in "Error Log")
只想到用一个变量实时刷新最小值，但 pop 之后就无法找回历史记录的前一个最小值了。
**易错点：** 在同步入栈操作时，最小栈由于可能遇到相同最小值，因此入栈辅助栈的判断条件必须是 `<=`，避免重复最小值在常规栈出栈时被误删。

### 3. 核心解法：辅助栈 / 差值法常数优化
**3.1 思路分析**
**破局点 1 (辅助栈)：** 利用一个**辅助栈**与常规栈同步进出，永远将当前层的历史最小值保存在上面。
- 常规栈正常压入弹出；最小栈 push 小于等于当前栈顶的元素；出栈时如果正好是当前最小元素，最小栈同步剥离。
**破局点 2 (差值空间优化)：** 栈内不存真实值，而是存**当前值与历史最小值的差值 (`diff`)**。出栈时一旦遇到负差值，说明正在弹出的就是极小值边界，此时做减法即可完美倒推并还原上一步的最小值。

**3.2 代码实现 (C++)**
```C++ 辅助栈
class MinStack {
    stack<int> min_stack;
    stack<int> norm_stack;
public:
    MinStack() {
        min_stack.push(INT_MAX); // 避免边界空栈判断
    }
    void push(int val) {
        norm_stack.push(val);
        if(val <= min_stack.top()){ 
            min_stack.push(val);
        }
    }
    void pop() {
        int top = norm_stack.top();
        norm_stack.pop();
        if(top == min_stack.top()){
            min_stack.pop();
        }
    }
    int top() {
        return norm_stack.top();
    }
    int getMin() {
        return min_stack.top();
    }
};
```

```C++ 无辅助栈，常数优化
class MinStack {
    stack<long long> min_stack;
    long long min_value;
public:
    MinStack() {
        
    }
    
    void push(int val) {
        if(min_stack.empty()){
            min_value = val;
            min_stack.push(val - min_value);
        }else{ // 先压入diff，再更新最小值。diff为负说明当前为最小值。
            long long diff = val - min_value; // diff含义为当前值与【之前的】最小值的差
            min_stack.push(diff);
            if(diff < 0) min_value = val;
        }
    }
    
    void pop() {
        long long top_diff = min_stack.top();
        min_stack.pop();
        if(top_diff < 0){ // 当前元素是最小值，还原最小值为上一次的最小值
            min_value = min_value - top_diff;
        }
    }
    
    int top() {
        long long top_diff = min_stack.top();
        if(top_diff < 0){
            return min_value;
        }else{
            return (int)(top_diff + min_value);
        }
    }
    
    int getMin() {
        return (int)min_value;
    }
};
```

### 4. 复杂度分析
- **时间复杂度**: $O(1)$。所有基本入栈出栈及获取最值操作均基于栈定长计算。
- **空间复杂度**: $O(N)$（辅助栈法） / $O(1)$（差值空间优化法）。存差值法极大缩减了额外数据结构带来的常数内存负担。

### 5. 总结与反思
利用辅助栈存储“历史状态快照”可完美解决极值同步，而“差值覆盖法”则是降维打击般的极致空间优化！
