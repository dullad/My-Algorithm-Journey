# LeetCode 84. 柱状图中最大的矩形

### 1. 题目链接
[https://leetcode.cn/problems/largest-rectangle-in-histogram/description/?envType=study-plan-v2&envId=top-100-liked]

### 2. 我的初次思路与遇到的问题 (The Error in "Error Log")
找到每个柱子左右边第一个小于其的柱子。

### 3. 核心解法：单调栈
**3.1 思路分析**
**破局点：** 巧妙使用**单调递增栈**记录相对落差。遍历每个元素，如果目前遍历的短于栈中高度，证明栈顶的长板遇到了强硬的“右护法遮挡”可以收网截断。而左边界也因栈的递推特性正好处于此时新的栈顶。

**3.2 代码实现 (C++)**
```C++
class Solution {
public:
    int largestRectangleArea(vector<int>& heights) {
        // 找 右侧第一个比自身小的元素 及 左侧第一个比自身小的元素下标
        int n = heights.size();
        vector<int> left(n, -1), right(n, n);

        stack<int> my_st; // 递增栈
        for(int i = 0; i < n; i++){
            while(!my_st.empty() && heights[my_st.top()] >= heights[i]){
                right[my_st.top()] = i;
                my_st.pop();
            }
            if(!my_st.empty()) left[i] = my_st.top(); // 栈顶比当前值小
            my_st.push(i);
        }
        int ans = 0;
        for(int i = 0; i < n; i++){
            ans = max((right[i] - left[i] - 1) * heights[i], ans);
        }
        return ans;
    }
};
```

### 4. 复杂度分析
- **时间复杂度**: $O(N)$。每个木板坐标都会进栈、出栈最多一次，两次遍历过程十分线性。
- **空间复杂度**: $O(N)$。单调栈空间与维护左右极限短板的等长数组辅助。

### 5. 总结与反思
利用单调栈可以在近乎单趟的遍历过程中解决左右双向探测边界的问题。
