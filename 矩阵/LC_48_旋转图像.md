### 2. 我的初次思路与遇到的问题 (The Error in "Error Log")

*在这里详细记录你最开始的想法，哪怕是错误的。*
由外向内，一层一层旋转。
每一层选取左上角的 `(i, i)` 到右上角的 `(i, n-1-i)` 这段区间的元素，进行 **四角轮换**。
- `temp` 保存左上角。
- 左上角 $\leftarrow$ 左下角
- 左下角 $\leftarrow$ 右下角
- 右下角 $\leftarrow$ 右上角
- 右上角 $\leftarrow$ `temp` (原左上角)
这样轮换只需要一个中间变量。

### 3. 核心解法：原地旋转

**3.1 思路分析**

进阶方法：旋转90度 = 水平翻转+主对角线翻转
1. 水平上下翻转：`row[i]` 和 `row[n-1-i]` 交换
2. 主对角线翻转（转置）：`(i, j)` 和 `(j, i)`交换
这种方法写起来块，不容易写错坐标逻辑。 

**3.2 代码实现 (C++)**

```c++
class Solution {
public:
    void rotate(vector<vector<int>>& matrix) {
        int k = 0;
        int n = matrix.size();
        while(k < n / 2){
            for(int i = k; i < n - 1 - k; i++){
                // 保存左上角
                int temp = matrix[k][i];
                
                // 1. 左上角 = 左下角
                matrix[k][i] = matrix[n - 1 - i][k];
                
                // 2. 左下角 = 右下角
                matrix[n - 1 - i][k] = matrix[n - 1 - k][n - 1 - i];
                
                // 3. 右下角 = 右上角
                matrix[n - 1 - k][n - 1 - i] = matrix[i][n - 1 - k];
                
                // 4. 右上角 = 原左上角
                matrix[i][n - 1 - k] = temp;
            }
            k++;
        }
    }
};
```

```C++
class Solution {
public:
    void rotate(vector<vector<int>>& matrix) {
        int n = matrix.size();
        
        // 1. 上下水平翻转
        // 也可以用 reverse(matrix.begin(), matrix.end());
        for (int i = 0; i < n / 2; ++i) {
            for (int j = 0; j < n; ++j) {
                swap(matrix[i][j], matrix[n - 1 - i][j]);
            }
        }
        
        // 2. 主对角线对称翻转 (Transpose)
        for (int i = 0; i < n; ++i) {
            // 注意 j 从 i+1 开始，只遍历对角线右上方，避免重复交换复原
            for (int j = i + 1; j < n; ++j) {
                swap(matrix[i][j], matrix[j][i]);
            }
        }
    }
};
```

### 4. 复杂度分析

- **时间复杂度**: $O(N^2)$。
- **空间复杂度**: $O(1)$。

### 5. 总结与反思

- 顺时针 90°：先 上下翻转，再 转置。
- 逆时针 90°：先 左右翻转，再 转置。
- 旋转 180°：先上下翻转，再左右翻转（或者旋转两次 90度）。
