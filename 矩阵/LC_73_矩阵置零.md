### 3. 核心解法：原地标记（利用首行首列）

**3.1 思路分析**

1.  **助空间法 ($O(M+N)$)**：
    -   用两个布尔数组记录哪些行、哪些列需要置零，空间复杂度是 $O(M+N)$。

2.  **原地标记法 ($O(1)$)**：
    -   关键点：**把矩阵的第一行和第一列作为“标志位”数组**，代替 `row_set` 和 `col_set`。
    -   **冲突问题**：`matrix[0][0]` 既属于第一行也属于第一列。如果它变为了 0，我们不知道是第一行有 0 还是第一列有 0。
    -   **解决方案**：
        -   先用两个额外的变量 `flag_col0` 和 `flag_row0` 记录第一列和第一行原本是否包含 0。
        -   然后遍历除第一行第一列外的剩余部分 (从 `[1][1]` 开始)，如果 `matrix[i][j] == 0`，就将对应的行首 `matrix[i][0]` 和列首 `matrix[0][j]` 置为 0。
        -   根据行首和列首的标记，将对应的 **内部** 行和列置零。
        -   最后，根据 `flag_col0` 和 `flag_row0` 单独处理第一行和第一列。

**3.2 代码实现 (C++)**

```c++
class Solution {
public:
    void setZeroes(vector<vector<int>>& matrix) {
        vector<int> row_set(matrix.size(), 0);
        vector<int> col_set(matrix[0].size(), 0);
        for(int i = 0; i < matrix.size(); i++){
            for(int j = 0; j < matrix[0].size(); j++){
                if(matrix[i][j] == 0){
                    row_set[i] = 1;
                    col_set[j] = 1;
                }
            }
        }
        for(int row = 0; row < matrix.size(); row++){
            if(row_set[row]){
                for(int j = 0; j < matrix[0].size(); j++){
                    matrix[row][j] = 0;
                }
            }
        }
        for(int col = 0; col < matrix[0].size(); col++){
            if(col_set[col]){
                for(int i = 0; i < matrix.size(); i++){
                    matrix[i][col] = 0;
                }
            }
        }
    }
};
```

```C++
class Solution {
public:
    void setZeroes(vector<vector<int>>& matrix) {
        int m = matrix.size();
        int n = matrix[0].size();
        bool flag_col0 = false;
        bool flag_row0 = false;

        // 1. 检查第一列是否有 0
        for (int i = 0; i < m; i++) {
            if (matrix[i][0] == 0) {
                flag_col0 = true;
                break;
            }
        }

        // 2. 检查第一行是否有 0
        for (int j = 0; j < n; j++) {
            if (matrix[0][j] == 0) {
                flag_row0 = true;
                break;
            }
        }

        // 3. 使用第一行和第一列作为标记位
        // 从 (1,1) 开始遍历内部
        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                if (matrix[i][j] == 0) {
                    matrix[i][0] = 0;
                    matrix[0][j] = 0;
                }
            }
        }

        // 4. 根据标记置零 (注意：依然只处理 i>0, j>0 的部分，防止过早覆盖第一行列)
        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                if (matrix[i][0] == 0 || matrix[0][j] == 0) {
                    matrix[i][j] = 0;
                }
            }
        }

        // 5. 最后单独处理第一行和第一列
        if (flag_row0) {
            for (int j = 0; j < n; j++) {
                matrix[0][j] = 0;
            }
        }
        if (flag_col0) {
            for (int i = 0; i < m; i++) {
                matrix[i][0] = 0;
            }
        }
    }
};
```

### 4. 复杂度分析

- **时间复杂度**: $O(MN)$。
- **空间复杂度**: $O(1)$。

### 5. 总结与反思

- **空间优化技巧**: 在矩阵问题中，如果需要记录每行/每列的状态，经常可以利用矩阵的首行或首列作为状态存储区，从而省去$O(M+N)$的额外空间。
- **边界处理**: 利用首行首列时，最大的坑在于`(0,0)`位置的复用。它既代表第一行的状态，也代表第一列的状态。为了避免混淆，一定要先把第一行和第一列的原始状态单独拿变量存起来，最后再处理它们。
