# LeetCode 121. 买卖股票的最佳时机

### 1. 题目链接
[https://leetcode.cn/problems/best-time-to-buy-and-sell-stock/description/?envType=study-plan-v2&envId=top-100-liked]

### 2. 我的初次思路与遇到的问题 (The Error in "Error Log")
维护一个当天前的最低价格，与一个最大获利值。获得每天的价格后，最大获利值更新为：当天价格-当天前的最低价格。最低价格更新为 今日价格 与 当天前价格 的最小值。
**易错点：** 计算最高点和最低点差值时，容易忽略时间先后的约束：买入操作必须在卖出之前发生。

### 3. 核心解法：贪心（一次遍历）
**3.1 思路分析**
**破局点：** 我们只需要关注在当前天数之前出现的最低价格。因此，使用一个变量记录历史最低价，并在遍历时，计算以当前价格卖出能获得的最大利润（当前价格减去历史最低价），以此更新全局最大利润值。

**3.2 代码实现 (C++)**
```c++
// 完整写出
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        if(prices.empty()) return 0;
        int min_p = prices[0], max_profit = 0;
        for(int i = 1; i < prices.size(); i++){
            max_profit = max(max_profit, prices[i] - min_p);
            min_p = min(min_p, prices[i]);
        }
        return max_profit;
    }
};
```

### 4. 复杂度分析
- **时间复杂度**: $O(N)$。遍历一次数组。
- **空间复杂度**: $O(1)$。仅使用了两个辅助变量。

### 5. 总结与反思
本题的核心在于遍历过程中维护历史最低价格状态，通过一次遍历同时更新最低价和最大利润。
