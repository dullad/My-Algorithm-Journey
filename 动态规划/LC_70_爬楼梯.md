# LeetCode 70. 爬楼梯

### 1. 题目链接
[https://leetcode.cn/problems/climbing-stairs/description/?envType=study-plan-v2&envId=top-100-liked]

### 2. 我的初次思路与遇到的问题 (The Error in "Error Log")
爬到当前级的方法=爬到前一级的方法+爬到前2级的方法。
*【思路梳理】：思路正确。*
**易错点：** 初始边界条件设置不当，容易在 `n=1` 或 `n=0` 时发生越界或逻辑错误。

### 3. 核心解法：动态规划（状态压缩）
**3.1 思路分析**
**破局点：** 将问题抽象为斐波那契数列。当前状态仅依赖于前两个状态，因此不需要维护完整的数组，只需使用两个变量交替步进更新即可。

**3.2 代码实现 (C++)**
```c++
// 完整写出
class Solution {
public:
    int climbStairs(int n) {
        int a = 1, b = 1;
        int ans = 0;
        if(n <= 1) return 1;
        for(int i = 2; i <= n; i++){
            ans = a + b;// 爬到前一级的方法+爬到前2级的方法
            a = b;
            b = ans;
        }
        return ans;
    }
};
```

### 4. 复杂度分析
- **时间复杂度**: $O(N)$。需要循环遍历依次计算到第 n 项。
- **空间复杂度**: $O(1)$。依靠常数个状态变量即可推演全局。

### 5. 总结与反思
利用滚动变量优化空间的一维状态转移，是状态压缩最朴素的基础形态。
