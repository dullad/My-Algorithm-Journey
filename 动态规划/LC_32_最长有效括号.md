# LeetCode 32. 最长有效括号

### 1. 题目链接
[https://leetcode.cn/problems/longest-valid-parentheses/description/?envType=study-plan-v2&envId=top-100-liked]

### 2. 我的初次思路与遇到的问题 (The Error in "Error Log")
dp[i]：以s[i]结尾(必须包含)的最长有效括号序列长度。如果是左括号，dp[i]必定为0，如果是右括号，那么看前一个字符如果是'('，那么dp[i] = dp[i - 2] + 2；否则如果是右括号，则要去找上一个未匹配的位置s[i - dp[i - 1] - 1]...
*【思路梳理】：思路正确且详实，极其清晰地剖分出了相邻闭合与外包闭合嵌套的问题核心。*
**易错点：** 当识别到可以跨层包含发生匹配拼接成形后（也就是发生长距连接成为 `"(...)"` 时），很可能忘记需要往之前更前段再加一次其自身的衔接有效长度（叠加前置的零散串连块），进而漏算拼接分块引发局部短切。

### 3. 核心解法：动态规划
**3.1 思路分析**
**破局点：** 重点在于讨论 `)`。将其收拢状态分为两层传递，一种是连续并排出现的闭合直接向左跨二阶合并，另一种是寻找囊内最长子括号组前边是否遗留着能够吞下自身的有效左弦，以此产生大幅向左跃进收编的操作。

**3.2 代码实现 (C++)**
```c++ 动态规划
// 看题解复现
class Solution {
public:
    int longestValidParentheses(string s) {
        int n = s.length();
        vector<int> dp(n, 0);// 以s[i]结尾(必须包含)的最长有效括号序列长度
        if(n <= 1) return 0;
        if(n > 1 && s[0] == '(' && s[1] == ')') dp[1] = 2;
        int ans = dp[1];
        for(int i = 2; i < n; i++){
            if(s[i] == '(') dp[i] = 0;
            else{
                if(s[i - 1] == '('){
                    dp[i] = dp[i - 2] + 2;
                }else{
                    if(i - dp[i - 1] - 1 >= 0 && s[i - dp[i - 1] - 1] == '('){
                        if(i - dp[i - 1] - 2 >= 0) dp[i] = dp[i - dp[i - 1] - 2] + dp[i - 1] + 2;
                        else dp[i] = dp[i - 1] + 2;
                    } 
                    else{
                        dp[i] = 0;
                    }
                    
                }
                ans = max(dp[i], ans);
            }
        }
        return ans;
    }
};
```

### 4. 复杂度分析
- **时间复杂度**: $O(N)$。仅仅完整顺着单字字符串迭代穿透比对了一整趟循环。
- **空间复杂度**: $O(N)$。依赖于等价宽度的状态记忆追踪横杠。

### 5. 总结与反思
针对尾部形态剥离分类处理，并依托历史积累连点成线，有效括号也成为串连逻辑的最佳推演场。
