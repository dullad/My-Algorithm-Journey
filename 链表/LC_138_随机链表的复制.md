# LeetCode 138. 随机链表的复制

### 1. 题目链接

[LeetCode 138. 随机链表的复制](https://leetcode.cn/problems/copy-list-with-random-pointer/description/?envType=study-plan-v2&envId=top-100-liked)

### 2. 我的初次思路与遇到的问题

初次思路：
- 第一遍建立原节点到新节点的映射。
- 第二遍连接新链表的 `next` 与 `random`。


### 3. 核心解法

#### 3.1 哈希表法（推荐先掌握）

**思路**：
- 用 `unordered_map<Node*, Node*> copy_map` 保存映射。
- 第二遍时通过映射 O(1) 找到复制节点并连边。

```cpp
class Solution {
public:
    Node* copyRandomList(Node* head) {
        unordered_map<Node*, Node*> copy_map;
        Node* cur = head;
        while(cur != NULL){
            // 第一次遍历创建映射
            Node* copy_node = new Node(cur->val);
            copy_map.insert({cur, copy_node});
            cur = cur->next;
        }
        cur = head;
        while(cur != NULL){
            // 第二次遍历把每个节点的next和random连起来
            copy_map[cur]->next = copy_map[cur->next];
            copy_map[cur]->random = copy_map[cur->random];
            cur = cur->next;
        }
        return copy_map[head];
    }
};
```

### 4. 复杂度分析

- **哈希表法**: 时间 $O(n)$，空间 $O(n)$。
