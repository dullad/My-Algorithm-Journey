# LeetCode 226. 翻转二叉树

### 1. 题目链接

[LeetCode 226. 翻转二叉树](https://leetcode.cn/problems/invert-binary-tree/description/?envType=study-plan-v2&envId=top-100-liked)

### 2. 核心解法：结构对调递归

**思路分析**
- 翻转的核心目标即是以各个节点为基准点，将其左右子树做互换对换。
- 比较好的解题遍历顺序是：对于当前节点，先交换他的左右孩子，然后对它的左端递归交换处理、对其右端递归交换处理，即**前序或者后序遍历**。注意不能用纯血的中序遍历解决，可能会引出指针重复复原的麻烦。

**代码实现 (C++)**

```cpp
class Solution {
public:
    TreeNode* invertTree(TreeNode* root) {
        if(!root) return root;
        // 交换当前节点的左右子节点
        TreeNode* temp = root->right;
        root->right = root->left;
        root->left = temp;
        // 分别递归操作原先两侧的枝叶
        invertTree(root->left);
        invertTree(root->right);
        return root;
    }
};
```

### 3. 复杂度分析

- **时间复杂度**: (N)$。节点互换，无多余过程浪费。
- **空间复杂度**: (H)$。递归堆栈占用，通常等同于树的高。

### 4. 总结与反思

- 此题操作属于改变结构的修改，理解交换是发生在引用之间的概念最为重要（
oot->left 本身存的就是其子树跟节点的地址），调换他们就等于带连着下面的所有分脉络被对向置换完成了。
