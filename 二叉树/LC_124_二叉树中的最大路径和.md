# LeetCode 124. 二叉树中的最大路径和

### 1. 题目链接

[LeetCode 124. 二叉树中的最大路径和](https://leetcode.cn/problems/binary-tree-maximum-path-sum/description/?envType=study-plan-v2&envId=top-100-liked)

### 2. 我的初次思路与遇到的问题
```cpp
class Solution {
    int maxFromRoot(TreeNode* root){
        // 从根出发的最大路径
        if(!root) return 0;
        return max(root->val, max(maxFromRoot(root->left), maxFromRoot(root->right)) + root->val);
    }
public:
    int maxPathSum(TreeNode* root) {
        // 所有情况的最大路径和
        if(!root) return 0;
        // 包含根的最大路径和，与不包含根的最大路径和比大小
        // 不包含根的最大路径和，为左右子树的所有情况的最大路径和的最大值。
        int maxSon = -1001;
        if(root->left){
            int leftMax = maxPathSum(root->left);
            if(leftMax > maxSon) maxSon = leftMax;
        }
        if(root->right){
            int rightMax = maxPathSum(root->right);
            if(rightMax > maxSon) maxSon = rightMax;
        }
        int maxInclude = root->val;
        int a = maxFromRoot(root->left);
        int b = maxFromRoot(root->right);
        if(a > 0) maxInclude += a;
        if(b > 0) maxInclude += b;
        return max(maxInclude, maxSon);
    }
};
```
- 直接把解决解法生生拆成了 包含根的最大路径和，与不包含根的最大路径和比大小。然后包含根的最大路径和，又写了一层DFS递归搜索。最终写了个二次重复套叠的两重 DFS 树的递归，严重让复杂度越界飙升到 $O(N^2)$。

### 3. 核心解法：后序遍历“夹带私货”更新最大值

**思路分析**
- 这和求解二叉树的直径（LC_543）思路如出一辙，对于任意处于二叉树其中一段连续路径链路的过程而言，其一定是存在并且具有一个所谓折弯“最高点”转点（根源头）节点的。
- 这个所谓属于自身范围的最优横跨全路线解必定满足： `本身基准值 + 左分支侧的路径和最大正数 + 右分支侧的路径和最大正数`。
- DFS规定： **返回当前节点连续向下走的最大路径和**（如果是负的可以割除，强加上一个 `max(0, xxx)`）。
- 分别搜索左右子树的增益（最大路径和 与 0 的最大值），经过当前root的最大路径和应该就是`root->val + leftMax + rightMax`，与`maxSum`全局最大路径和和比较。

**代码实现 (C++)**

```cpp
class Solution {
    int maxSum = INT_MIN; 
    int dfs(TreeNode* root){
        // 返回从根节点向下出发的最大路径和
        if(!root) return 0;
        // 分别收集索取下方两条支的最大收益收益 
        int leftMax = max(dfs(root->left), 0);
        int rightMax = max(dfs(root->right), 0);
        // 经过root的最大路径和与全局最大路径和比较
        maxSum = max(root->val + leftMax + rightMax, maxSum);
        return max(leftMax, rightMax) + root->val;
    }
public:
    int maxPathSum(TreeNode* root) {
        dfs(root);
        return maxSum;
    }
};
```

### 4. 复杂度分析

- **时间复杂度**: $O(N)$。单层后序遍历的同时做全局最优判断。
- **空间复杂度**: $O(H)$。递归回溯压栈深度受到其高度控制。

### 5. 总结与反思

- 维护一个全局的极大值，然后在后序遍历中用当前最大值与全局最大值比较以维护全局最大值。
- `max(0, 某个子树DFS)` 的精华剪枝是这题与其它最长链路的最大不同之处。
