### 2. 我的初次思路与遇到的问题

### 3.1 核心解法1：上下边界法（DFS）

**思路分析**
在递归遍历的过程中，不仅要保证当前节点大于左子节点、小于右子节点，还要保证它所在的整棵左子树所有节点都小于它，整棵右子树所有节点都大于它。因此采用带有上下界的 DFS：向下传递有效区间 `[lower, upper]`，越界即返回 false。

```cpp
class Solution {
    bool dfs(TreeNode* root, long long lower, long long upper){
        if(root == nullptr) return true;
        if(root->val <= lower || root->val >= upper) return false;
        return dfs(root->left, lower, root->val) && dfs(root->right, root->val, upper);
    }
public:
    bool isValidBST(TreeNode* root) {
        return dfs(root, LLONG_MIN, LLONG_MAX);
    }
};
```

### 3.2 核心解法2：中序遍历严格升序

**思路分析**
二叉搜索树的中序遍历一定是一个严格递增的序列。我们在中序遍历的过程中，维护一个 `prev` 变量记录遍历到的上一个节点的值。如果判断当前节点的值小于等于 `prev`，说明不满足严格递增，返回 `false`。

**代码实现 (C++)**

```cpp
class Solution {
    long long prev = LLONG_MIN;
    bool isorder(TreeNode* root){
        if(root == nullptr) return true;
        if(!isorder(root->left)) return false;
        if(root->val <= prev) return false;
        prev = root->val;
        return isorder(root->right);
    }
public:
    bool isValidBST(TreeNode* root) {
        return isorder(root);
    }
};
```

### 4. 复杂度分析

- **时间复杂度**: `$O(N)$`。整树仅进行一次单向无回退DFS或中序巡回。
- **空间复杂度**: `$O(H)$`。对应回溯压栈深度，最好为 `$O(\log N)$`（平衡树），最差 `$O(N)$`。

### 5. 总结与反思

- 验证 BST 的两种核心思想：一是自顶向下传区间边界验证（前序/DFS），二是从小到大遍历比较邻项（中序遍历）。中序写法的全局变量法或引用法在树题中很常用，能省去很多传参复杂性。
