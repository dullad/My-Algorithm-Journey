# LeetCode 108. 将有序数组转换为二叉搜索树

### 1. 题目链接

[LeetCode 108. 将有序数组转换为二叉搜索树](https://leetcode.cn/problems/convert-sorted-array-to-binary-search-tree/description/?envType=study-plan-v2&envId=top-100-liked)

### 2. 核心解法：

**思路分析**
二叉搜索树的中序遍历是严格升序序列，而题目给定的正是严格升序数组。为了保证构造出高度平衡的二叉搜索树（左右子树高度差不超过1），我们需要每次选择数组的中间元素作为根节点，然后将数组分成左右两部分，递归地应用该逻辑构造左右子树，这样就能自然保证两边数量均匀匹配平衡要求。

**代码实现 (C++)**

```cpp
class Solution {
    TreeNode* buildBST(vector<int>& nums, int l, int r){
        if(l > r) return nullptr;
        int mid = (l + r) / 2;
        TreeNode* root = new TreeNode(nums[mid]);
        root->left = buildBST(nums, l, mid - 1);
        root->right = buildBST(nums, mid + 1, r);
        return root;
    }
public:
    TreeNode* sortedArrayToBST(vector<int>& nums) {
        return buildBST(nums, 0, nums.size() - 1); 
    }
};
```

### 3. 复杂度分析

- **时间复杂度**: `$O(N)$`，每个节点被访问一次。
- **空间复杂度**: 递归栈或迭代栈的深度，平均 `$O(\log N)$`，退化链表时最坏为 `$O(N)$`（本题因为强行取中点所以空间复杂度严格为 `$O(\log N)$`）。

### 4. 总结与反思

- 这道题是分治法的经典应用。利用 BST 的性质以及高度平衡的要求，每次取中点作为根节点。思路不难，主要是熟练掌握树的递归建立模板。
