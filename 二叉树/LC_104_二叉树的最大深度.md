# LeetCode 104. 二叉树的最大深度

### 1. 题目链接

[LeetCode 104. 二叉树的最大深度](https://leetcode.cn/problems/maximum-depth-of-binary-tree/description/?envType=study-plan-v2&envId=top-100-liked)

### 2. 核心解法：递归 DFS 与迭代 BFS

**思路分析**
1. **DFS (递归法)**：单层逻辑非常清晰，当前子树深度等于 左子树深度与右子树深度比大小后取最长 再加上它本身的 1 即可。属于由底向上的后序处理。
2. **BFS (层序遍历法)**：利用层序遍历的思路，一层一层的扩散出去，记录好 layer_num 的外循环每运转完整的一圈记录为加1即可。

**代码实现 (C++)**

**DFS (递归)**
```cpp
class Solution {
public:
    int maxDepth(TreeNode* root) {
        if(root == nullptr) return 0;
        return max(maxDepth(root->left), maxDepth(root->right)) + 1;
    }
};
```

**BFS (层序)**
```cpp
class Solution {
public:
    int maxDepth(TreeNode* root) {
        if(!root) return 0;
        queue<TreeNode*> q;
        q.push(root);
        int ans = 0;
        while(!q.empty()){
            int layer_num = q.size();
            while(layer_num){
                TreeNode* node = q.front();
                q.pop();
                if(node->left) q.push(node->left);
                if(node->right) q.push(node->right);
                layer_num--;
            }
            ans++;
        }
        return ans;
    }
};
```

### 3. 复杂度分析

- **时间复杂度**: `$O(N)$`。均需要遍历完整的树。
- **空间复杂度**: DFS需要深度为高度 `$(H)$` 的系统栈，BFS 需要高度为 `$(N)$`（最宽一层）的队列进行存储。

### 4. 总结与反思

- 递归做法极度浓缩，写起来快但如果深度过大容易栈溢出；BFS在计算深度上往往非常通用并更加稳定，属于各有利弊的基本题型。
