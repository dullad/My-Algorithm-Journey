### 3.1 核心解法1：广度优先搜索，层序遍历记录每一层最右节点

**思路分析**
利用队列进行标准的层序遍历，按层推进处理节点。每一层我们通过当前队列的大小 `layer_num` 控制遍历次数，当处理到一层的最后一个节点时（`layer_num == 0`），该节点就是从右侧看过去暴露出来的节点，将其值加入结果集中。

```cpp
class Solution {
public:
    vector<int> rightSideView(TreeNode* root) {
        // 层序遍历,取每一层最右边值
        vector<int> res;
        if(!root) return res;
        queue<TreeNode*> myque;
        myque.push(root);
        while(!myque.empty()){
            int layer_num = myque.size();
            while(layer_num){
                TreeNode* cur = myque.front();
                myque.pop();
                if(cur->left) myque.push(cur->left);
                if(cur->right) myque.push(cur->right);
                layer_num--;
                if(layer_num == 0) res.push_back(cur->val);
            }
        }
        return res;
    }
};
```

### 3.2 核心解法2：深度优先搜索，根右左，某个深度首次达到记录

**思路分析**
巧妙采用“根 -> 右 -> 左”的先序遍历变体。这样探索每一层时，必然是优先访问最右侧的节点。对于遍历到的每个节点，如果它所在的深度（`depth`）恰好等于 `res` 数组当前的大小，说明这是在该深度下我们遍历到的第一个（也就是最右边的）节点，收集答案即可。

**代码实现 (C++)**

```cpp
class Solution {
    vector<int> res;
    void dfs(TreeNode* node, int depth){
        if(!node) return;
        if(depth == res.size()) res.push_back(node->val);
        dfs(node->right, depth + 1);
        dfs(node->left, depth + 1);
    }
public:
    vector<int> rightSideView(TreeNode* root) {
        dfs(root, 0);
        return res;
    }
};
```

### 4. 复杂度分析

- **时间复杂度**: `$O(N)$`。整树最坏情况每个节点仅访问一次。
- **空间复杂度**: 最大 `$O(N)$`。层序遍历的队列大小最坏 $O(N)$；DFS回溯压栈深度最好为 `$O(\log N)$`，最差 `$O(N)$`。

### 5. 总结与反思

- 求层级特定视图（如右视图、左视图）有两种经典大思路：一是 BFS 层序遍历取尾元素（思路最直接），二是利用由右向左的 DFS 先序变体，将 `res.size()` 与 `depth` 做对比记录。理解第二种方法的剪枝与深度逻辑非常有助于提升图/树搜索技巧。
