# LeetCode 101. 对称二叉树

### 1. 题目链接

[LeetCode 101. 对称二叉树](https://leetcode.cn/problems/symmetric-tree/description/?envType=study-plan-v2&envId=top-100-liked)

### 2. 核心解法：递归判断镜像

**思路分析**
- 检查一棵树是否对称，等价于检查它的“左子树”与“右子树”是不是互相成镜像（互相反转）。
- 我们可以定义辅助函数 cmp(left, right) 接收一对子树节点：
  1. 同时为空则为镜像。
  2. 一个为空一个不为空，或者两者值不同，不是镜像。
  3. 若值相同，它是否是对称取决于其外侧子树 left->left & right->right 以及内侧子树 left->right & right->left 是否均为镜像。

**代码实现 (C++)**

```cpp
class Solution {
    bool cmp(TreeNode* left, TreeNode* right){
        if(!left && !right) return true; // 两个都是空节点
        else if(left && right && left->val == right->val){
            // 两个都非空节点，且值相同
            return cmp(left->left, right->right) && cmp(left->right, right->left); 
        }
        else{
            return false;
        }
    }
public:
    bool isSymmetric(TreeNode* root) {
        return cmp(root->left, root->right);
    }
};
```

### 3. 复杂度分析

- **时间复杂度**: `$O(N)$`，每次成对检查节点，最多遍历整个树的过程。
- **空间复杂度**: `$O(H)$`，即递归栈最多为主树的高度。

### 4. 总结与反思

- “分治法”判断对称二叉树：将一个大问题拆解为校验“左孩子的左孩子 与 右孩子的右孩子”（即外侧同向）、“左孩子的右孩子 与 右孩子的左孩子”（即内侧交叉）这两个小问题。
