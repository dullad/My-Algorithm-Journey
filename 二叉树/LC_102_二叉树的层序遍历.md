# LeetCode 102. 二叉树的层序遍历

### 1. 题目链接

[LeetCode 102. 二叉树的层序遍历](https://leetcode.cn/problems/binary-tree-level-order-traversal/description/?envType=study-plan-v2&envId=top-100-liked)

### 2. 核心解法：队列BFS实现批处理

**思路分析**
- 使用广度优先搜索 (BFS)，核心数据结构为 queue。
- 因为要求每一层的数据各自存为独立的数组 vector<int>，关键在于分离层数。
- 在开始处理某特定层之前，队列中存放的**恰好是该层所有节点**。故先记录下 size，然后用循环连续处理这么多节点，在此过程中它们被推出，而下一层节点不断添加到末尾。

**代码实现 (C++)**

```cpp
class Solution {
public:
    vector<vector<int>> levelOrder(TreeNode* root) {
        vector<vector<int>> res;
        if(!root) return res;
        queue<TreeNode*> q;
        q.push(root);
        while(!q.empty()){
            int layer_num = q.size();
            vector<int> layer_res;
            while(layer_num--){
                TreeNode* temp = q.front();
                layer_res.push_back(temp->val);
                q.pop();
                if(temp->left) q.push(temp->left);
                if(temp->right) q.push(temp->right);
            }
            res.push_back(layer_res);
        }
        return res;
    }
};
```

### 3. 复杂度分析

- **时间复杂度**: (N)$，每个节点进出队列各一次。
- **空间复杂度**: (N)$，对于一棵完全二叉树，队列最多的元素个数可能出现满的一层，规模达 /2$。

### 4. 总结与反思

- 基于统计按层分发的队列批处理应用极广。只要遇到图、网络或者树结构中需要关注层次关系的题目，此方法都是最佳选择模板。
