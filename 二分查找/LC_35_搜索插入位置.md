### 2. 我的初次思路与遇到的问题 (The Error in "Error Log")

*在这里详细记录你最开始的想法，哪怕是错误的。*
二分查找，没找到mid的左边或右边继续寻找。
易错点： 当循环结束（没有找到 target）时，到底该返回 left、right 还是 mid？这是初学者极其容易搞混的地方，经常需要靠代入具体数字盲猜。

### 3. 核心解法：哈希表

**3.1 思路分析**
破局点： 深刻理解 while(left <= right) 的终止条件。当循环退出时，必然满足 left > right，此时 left 指针指向的位置，永远是第一个大于目标值的位置，而这恰好就是目标值需要插入的正确位置。

**3.2 代码实现 (C++)**
```c++
class Solution {
public:
    int searchInsert(vector<int>& nums, int target) {
        int left = 0, right = nums.size() - 1;
        while(left <= right){
            int mid = (left + right) / 2;
            if(nums[mid] == target){
                return mid;
            }else if(nums[mid] > target){
                right = mid - 1;
            }else{
                left = mid + 1;
            }
        }
        return left;
    }
};
```

### 4. 复杂度分析

- **时间复杂度**: $O(log(N))$。
- **空间复杂度**: $O(1)$。

### 5. 总结与反思

- 基础二分查找的变体，牢记规律：while(left <= right) 结束时，left 即为第一个大于等于目标值的插入位置。
