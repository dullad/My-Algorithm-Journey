二维矩阵拉成一维矩阵，关键是二维坐标和一维位置的转换。

### 3. 核心解法：哈希表

**3.1 思路分析**
破局点： 因为每一行的第一个整数大于前一行的最后一个整数，整个矩阵首尾相连其实就是一个严格递增的一维数组。我们将其视为长度为 M * N 的一维数组进行二分，通过公式 row = mid / cols, col = mid % cols 将一维索引重新映射回二维矩阵的物理位置。

**3.2 代码实现 (C++)**
```c++
class Solution {
public:
    bool searchMatrix(vector<vector<int>>& matrix, int target) {
        int left = 0, right = matrix.size() * matrix[0].size() - 1;
        while(left <= right){
            int mid = (left + right) / 2;
            int row = mid / matrix[0].size(), col = mid % matrix[0].size();
            if(matrix[row][col] == target) return true;
            else if(matrix[row][col] < target){
                left = mid + 1;
            }else{
                right = mid - 1;
            }
        }
        return false;
    }
};
```

### 4. 复杂度分析

- **时间复杂度**: $O(log(M*N))$。
- **空间复杂度**: $O(1)$。

### 5. 总结与反思

- 将严格递增二维矩阵视作一维数组处理，掌握核心公式：row = mid / 列数, col = mid % 列数。
