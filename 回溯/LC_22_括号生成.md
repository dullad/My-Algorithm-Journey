# LeetCode 22. 括号生成

### 1. 题目链接
[22. 括号生成](https://leetcode.cn/problems/generate-parentheses/description/?envType=study-plan-v2&envId=top-100-liked)

### 2. 核心解法：条件限制回溯

**3.1 思路分析**
实质是在不断二选一（加左括号还是加右括号）的回溯树中加规则剪枝：
1. **可加左括号的条件**：左括号数量尚未达到 `n`。
2. **可加右括号的条件**：当前持有的右括号数量**必须少于**左括号的数量。这样才能保证生成的括号是合法的闭合！

**3.2 代码实现 (C++)**
```cpp
class Solution {
    vector<string> res;
    string a;
    void backtrack(int num_l, int num_r, int n){
        if(a.length() == n * 2){ // 总长度为2n时收集答案
            res.push_back(a);
            return;
        }
        // 左括号没用完，可以一直放左括号
        if(num_l < n){
            a.push_back('(');
            backtrack(num_l + 1, num_r, n);
            a.pop_back();
        }
        // 只有当前右括号少于左括号时，才能放右括号进行闭合
        if(num_r < num_l){
            a.push_back(')');
            backtrack(num_l, num_r + 1, n);
            a.pop_back();
        }
    }
public:
    vector<string> generateParenthesis(int n) {
        backtrack(0, 0, n);
        return res;
    }
};
```

### 4. 复杂度分析
- **时间复杂度**: `$O(\frac{4^n}{\sqrt{n}})$`。组合数学中的第 N 个卡特兰数规则。
- **空间复杂度**: `$O(n)$`。除了返回值空间，递归栈深度的上限恰好是 `2n`。

### 5. 总结与反思
- 掌握 **左侧随时放，右侧 `right < left` 时才放** 这一控制法，自然得到全合规组合。
