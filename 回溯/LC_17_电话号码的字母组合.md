### 3. 核心解法：回溯法

**3.1 思路分析**
标准的多个集合组合问题。`index` 控制递归的深度（代表遍历到第几个按键），`for` 循环控制单层的广度（按键对应的不同字母）。

**3.2 代码实现 (C++)**
```cpp
class Solution {
    unordered_map<char, string> my_map{
        {'2', "abc"}, {'3', "def"}, {'4', "ghi"}, 
        {'5', "jkl"}, {'6', "mno"}, {'7', "pqrs"}, 
        {'8', "tuv"}, {'9', "wxyz"}
    };
    vector<string> res;
    string path;
    void traceback(string digits, int index){
        if(index == digits.size()){
            res.push_back(path);
            return;
        }
        string letters = my_map[digits[index]];
        for(int j = 0; j < letters.length(); j++){
            path.push_back(letters[j]);
            traceback(digits, index + 1);
            path.pop_back();
        }
    }
public:
    vector<string> letterCombinations(string digits) {
        if(digits.empty()) return res;
        traceback(digits, 0);
        return res;
    }
};
```

### 4. 复杂度分析
- **时间复杂度**: `$O(3^m \times 4^n)$`，m和n分别是包含3/4个字母的按键总数。
- **空间复杂度**: `$O(m+n)$`，主要为递归栈调用的深度和系统保存临时变量。

### 5. 总结与反思
- 回溯模板：找准**树深度由谁控制**（本题是 `index`），**树宽度由谁控制**（本题是字典对应的字母串）。
